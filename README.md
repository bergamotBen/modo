# MODO

## A PWA todo app completely distinct from the Pomodoro Technique.

This little todos project is to help me learn:

- Serverless design patterns
- Microservices with Supabase edge functions
- PWA push notifications

...and keep my React skills sharp.

I'm trying to use some of the project management features of Git to make it clear what I'm working on.
